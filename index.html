📋 INTERVIEW SCRIPT — What to SAY during coding interviews

        
1. Clarify"Can I assume the array is sorted?" "Can there be duplicates?" "What's the expected output for empty input?"

        
2. Brute Force"Naive approach would be O(n²) — iterate all pairs. Let me optimize this."

        
3. Optimize"I notice we're redoing work — if I use a HashMap / window / prefix sum, I can get O(n)."

        
4. Code"Let me code this step by step. I'll handle edge cases at the end."

        
5. Test"Let me trace through [2,7,11,15] with target=9... yes, returns [0,1]. Looks correct."
